# Lesson 08: Fund Manager Due Diligence

## Learning Objectives
By the end of this lesson, you will be able to:
- Conduct comprehensive due diligence on mutual fund managers
- Evaluate investment process, team stability, and organizational culture
- Apply quantitative and qualitative assessment frameworks
- Assess manager skill versus luck using statistical analysis
- Create systematic manager monitoring and replacement procedures

## Introduction

Selecting the right mutual fund manager is often the most critical decision in active portfolio management. While past performance provides some insight, it represents only one piece of a complex puzzle. Effective due diligence requires evaluating investment philosophy, process consistency, team dynamics, and organizational support. This lesson provides a comprehensive framework for evaluating mutual fund managers and building sustainable investment partnerships.

## Due Diligence Framework

### The Four Pillars of Manager Evaluation

#### 1. Investment Philosophy and Process

**Investment Philosophy Assessment:**
```

1. Value Proposition
   - What unique value does the manager provide?
   - How does the approach differ from passive alternatives?
   - What market inefficiencies are being exploited?

2. Investment Style Consistency
   - Is the style clearly defined and maintained?
   - Are style drifts minimal and explainable?
   - How does the manager define their niche?

3. Risk Management Philosophy
   - How is risk defined and measured?
   - What controls prevent excessive risk-taking?
   - How are downside scenarios addressed?

## Advanced Due Diligence Techniques

### Statistical Significance Testing

#### Alpha Significance Analysis

**Performance Skill Assessment:**
```
Statistical Testing Framework:
1. Null Hypothesis: Alpha = 0 (no skill)
2. Alternative Hypothesis: Alpha ≠ 0 (skill present)
3. Test Statistic: t-statistic = Alpha / Standard Error
4. Significance Level: 5%

Example Calculation:
Fund Alpha: 2.1% annually
Standard Error: 0.8%
t-statistic: 2.625
Critical Value (95%): ±1.96
p-value: 0.009

Conclusion:
- t-statistic > critical value
- p-value < 0.05
- Reject null hypothesis
- Alpha statistically significant
- Evidence of manager skill
```

**Bootstrap Analysis:**
```
Bootstrap Methodology:
1. Resample monthly returns with replacement
2. Calculate alpha for each resample (10,000 iterations)
3. Determine confidence intervals
4. Assess probability of observed alpha

Bootstrap Results:
- Observed Alpha: 2.1%
- 95% Confidence Interval: 0.8% to 3.4%
- Probability Alpha > 0: 99.2%
- Probability Alpha > 1%: 94.5%

Interpretation:
- High confidence alpha is positive
- Strong evidence of skill
- Robust across different samples
```

#### Performance Persistence Testing

**Persistence Analysis:**
```
Rolling Window Analysis:
3-Year Rolling Alphas:
2019-2021: +1.8%
2020-2022: +2.3%
2021-2023: +2.0%

Persistence Metrics:

## Key Takeaways

### Essential Due Diligence Principles

1. **Due diligence is multi-dimensional** - performance, process, team, and operations must all be evaluated comprehensively

2. **Past performance provides limited insight** - focus on understanding why performance was generated rather than simply measuring results

3. **Process consistency is crucial** - look for disciplined adherence to stated investment approach and risk management

4. **Team stability and succession planning** - organizational health and continuity are critical for long-term success

5. **Operational excellence is foundational** - even the best investment process fails without robust operational infrastructure
