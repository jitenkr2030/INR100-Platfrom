# Lesson 134: Performance Attribution Analysis

## Understanding Performance Attribution

Performance attribution analysis decomposes portfolio returns to identify sources of performance:
- **Asset Allocation**: Strategic allocation decisions
- **Security Selection**: Individual stock picking
- **Interaction Effects**: Combined allocation and selection
- **Market Timing**: Tactical allocation changes

## Attribution Models

### Brinson Model
#### Core Components
- **Allocation Effect**: Returns from over/underweighting asset classes
- **Selection Effect**: Returns from security selection within asset classes
- **Interaction Effect**: Combined allocation and selection decisions

#### Formula Structure
```
Portfolio Return - Benchmark Return = Allocation Effect + Selection Effect + Interaction Effect
```

### Performance Attribution Process
#### Step 1: Establish Benchmark
- **Strategic Benchmark**: Long-term target allocation
- **Tactical Benchmark**: Adjusted allocation based on timing
- **Custom Benchmark**: Match portfolio composition

#### Step 2: Calculate Component Returns
- **Asset Class Returns**: Individual asset class performance
- **Security Returns**: Individual security performance
- **Weight Analysis**: Beginning, ending, and average weights

#### Step 3: Attribution Calculations
- **Allocation Attribution**: (Portfolio Weight - Benchmark Weight) × Asset Return
- **Selection Attribution**: Portfolio Weight × (Security Return - Asset Return)
- **Interaction Attribution**: (Portfolio Weight - Benchmark Weight) × (Security Return - Asset Return)

## Asset Allocation Attribution

### Strategic Allocation Analysis
#### Long-term Allocation Decisions
- **Target Allocation**: Policy portfolio weights
- **Strategic Drift**: Changes due to market movements
- **Rebalancing Impact**: Effect of rebalancing activities
- **Strategic Timing**: Long-term allocation timing decisions

#### Tactical Allocation Analysis
#### Short-term Allocation Changes
- **Market Timing**: Over/underweighting based on market views
- **Economic Views**: Allocation based on economic forecasts
- **Valuation Timing**: Asset allocation based on valuations
- **Risk Management**: Allocation adjustments for risk control

### Allocation Metrics
#### Return Attribution
- **Excess Return**: Portfolio return minus benchmark return
- **Allocation Contribution**: Return from allocation decisions
- **Timing Contribution**: Return from timing decisions

#### Risk Attribution
- **Volatility Contribution**: Risk contribution by asset class
- **Correlation Impact**: Diversification benefits/losses
- **Tracking Error**: Deviation from benchmark risk

## Security Selection Attribution

### Within-Asset-Class Analysis
#### Equity Selection
- **Sector Allocation**: Over/underweighting sectors
- **Stock Selection**: Individual stock picking
- **Style Allocation**: Growth vs value tilts
- **Size Allocation**: Large vs small cap tilts

#### Fixed Income Selection
- **Duration Management**: Interest rate positioning
- **Credit Selection**: Investment grade vs high yield
- **Sector Selection**: Government vs corporate bonds
- **Maturity Selection**: Short vs long-term bonds

### Selection Metrics
#### Alpha Generation
- **Security Alpha**: Return above asset class benchmark
- **Selection Skill**: Consistent outperformance
- **Information Ratio**: Alpha per unit of tracking error

#### Risk-Adjusted Performance
- **Sharpe Ratio**: Return per unit of total risk
- **Information Ratio**: Return per unit of tracking error
- **Sortino Ratio**: Return per unit of downside risk

## Performance Measurement

### Return Metrics
#### Absolute Returns
- **Total Return**: Capital appreciation + income
- **Annualized Return**: Compound annual growth rate
- **Rolling Returns**: Returns over multiple periods
- **Risk-Adjusted Return**: Return per unit of risk

#### Relative Returns
- **Excess Return**: Portfolio return minus benchmark
- **Tracking Error**: Standard deviation of excess returns
- **Information Ratio**: Excess return divided by tracking error
- **Alpha**: Risk-adjusted excess return

### Risk Metrics
#### Volatility Measures
- **Standard Deviation**: Total portfolio volatility
- **Downside Deviation**: Volatility of negative returns
- **Maximum Drawdown**: Largest peak-to-trough decline
- **Value at Risk (VaR)**: Potential loss at given confidence level

#### Risk-Adjusted Metrics
- **Sharpe Ratio**: (Return - Risk-free rate) / Standard deviation
- **Sortino Ratio**: (Return - Risk-free rate) / Downside deviation
- **Calmar Ratio**: Return / Maximum drawdown
- **Sterling Ratio**: 3-year return / Average drawdown
