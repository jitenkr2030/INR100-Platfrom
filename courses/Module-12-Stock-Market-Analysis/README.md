# Module 12: Stock Market Analysis (Lessons 91-102)

## Overview
This intermediate-level module teaches comprehensive stock analysis techniques. Students will learn technical analysis, fundamental analysis, ratio analysis, and advanced evaluation methods used by professional analysts and investors.

## Learning Objectives
- Master technical analysis fundamentals
- Learn comprehensive ratio analysis
- Understand earnings and financial analysis
- Develop stock evaluation skills
- Learn about market microstructure

91. **Lesson 91**: Technical Analysis Basics
92. **Lesson 92**: Ratio Analysis Deep Dive

94. **Lesson 94**: Sector-Specific Analysis
95. **Lesson 95**: Management Quality Assessment
96. **Lesson 96**: Competitive Advantage Analysis
97. **Lesson 97**: Financial Modeling Basics
98. **Lesson 98**: Advanced Valuation Techniques
99. **Lesson 99**: Mergers and Acquisitions Analysis
100. **Lesson 100**: Market Microstructure and Algorithmic Trading

102. **Lesson 102**: Insurance and Risk Management

## Prerequisites
- Completion of Modules 1-11 (Foundation and Intermediate levels)
- Basic understanding of financial statements
- Familiarity with stock markets
