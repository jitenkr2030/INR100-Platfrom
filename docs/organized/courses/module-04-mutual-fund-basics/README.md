# Module 4: Mutual Fund Basics
## India's Most Popular Investment Vehicle (Lessons 36-55)

### Course Overview
Mutual funds are the gateway to wealth creation for millions of Indians. This module will transform you from a complete beginner to a confident mutual fund investor, understanding everything from NAV to SIP strategies.

### Learning Objectives
By completing this module, you will:
- Understand what mutual funds are and how they work
- Learn to read and interpret key mutual fund metrics
- Master different types of mutual funds and their purposes
- Develop a strategy for building wealth through mutual funds
- Learn how to select and monitor mutual fund investments
