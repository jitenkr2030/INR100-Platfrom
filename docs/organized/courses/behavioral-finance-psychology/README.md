## Key Concepts Covered

### Cognitive Biases
- **Confirmation Bias**: Seeking only supportive information
- **Overconfidence**: Overestimating investment abilities
- **Anchoring**: Using irrelevant reference points
- **Loss Aversion**: Feeling losses more than equivalent gains
- **Recency Bias**: Overweighting recent events
- **Herding**: Following crowd behavior

### Emotional Factors
- **Fear**: Market panic and risk avoidance
- **Greed**: Return expectations and risk-taking
- **Hope**: False optimism and magical thinking
- **Regret**: Learning from past decisions
- **Social Comparison**: Envy and status anxiety

### Decision-Making Tools
- **Investment Policy Statement**: Systematic approach
- **Decision Checklists**: Bias-proof evaluation
- **Emotional Management**: Stress reduction techniques
- **Automation Strategies**: Removing emotion from routine decisions
